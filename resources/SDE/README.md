# SDE

外资企业的软件开发相关岗位（SDE/SRE/DS/QA）的面试主要由两部分组成：考察代码能力的Code Challenge，aka白班写代码（这个就不说了，吃饭的家伙，基本功）；还有一个是考察以往团队合作经验以及和企业文化匹配程度的行为面试（Behavior Question，BQ）。
