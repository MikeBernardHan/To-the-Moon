很多日企是苹果重要的供应链成员，并且iPhone在日本的占有率是压倒性的多数。按理来说，苹果应该会在日本有规模不小的子公司。但没听说过成规模的校招计划，之前放出来几个实习岗位，但没听说过谁拿到了。不过是有听说过有学长毕业直接去苹果的。
